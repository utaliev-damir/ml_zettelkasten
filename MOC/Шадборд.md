# Дашборд ШАД

## 🔁 Задачи для повторения

```dataview
TABLE раздел, формат, сложность, дата_решения
FROM "Задачи"
WHERE для_повтора = true
SORT дата_решения ASC
```

## ❌ Задачи, решённые с подсказкой

```dataview
TABLE раздел, формат, сложность, источник
FROM "Задачи"
WHERE решена_сам = false AND дата_решения != null
SORT дата_решения DESC
```

## 🆕 Последние задачи

```dataview
TABLE раздел, формат, сложность
FROM "Задачи"
SORT file.ctime DESC
LIMIT 10
```

## 📊 Статистика по разделам

```dataview
TABLE length(rows) AS "Задач"
FROM "Задачи"
GROUP BY раздел
SORT length(rows) DESC
```

## 📈 Ритм решения за последние 30 дней

```dataview
TABLE length(rows) AS "Задач"
FROM "Задачи"
WHERE дата_решения >= date(today) - dur(30 days)
GROUP BY дата_решения
SORT дата_решения DESC
```

# Аудит знаний

## Методы без задач

Методы, которые ты знаешь теоретически, но не отработал.

```dataview
LIST
FROM "Методы"
WHERE !ключевые_задачи OR length(ключевые_задачи) = 0
SORT file.name
```

## Сигналы без методов

Сигналы, у которых не указан ни один метод.

```dataview
LIST
FROM "Сигналы"
WHERE !методы OR length(методы) = 0
SORT file.name
```

## Теоремы без следствий и связей

Теоремы, у которых не указаны методы или задачи, где они применяются.

```dataview
LIST
FROM "Теоремы"
WHERE (!методы OR length(методы) = 0) AND (!задачи OR length(задачи) = 0)
SORT file.name
```

## Определения без свойств и теорем

Определения-сироты, которые ни на что не сослались.

```dataview
LIST
FROM "Определения"
WHERE (!свойства OR length(свойства) = 0) AND (!теоремы OR length(теоремы) = 0)
SORT file.name
```

## Свойства без доказательства

```dataview
LIST
FROM "Свойства"
WHERE !contains(file.outlinks, this.file.link) AND length(file.outlinks) < 2
SORT file.name
```

## Алгоритмы без задач

```dataview
LIST
FROM "Алгоритмы"
WHERE !задачи OR length(задачи) = 0
SORT file.name
```

## Структуры данных без алгоритмов

```dataview
LIST
FROM "Структуры и типы данных"
WHERE !алгоритмы OR length(алгоритмы) = 0
SORT file.name
```

# Покрытие тем

## Распределение задач по формату

```dataview
TABLE length(rows) AS "Задач"
FROM "Задачи"
GROUP BY формат
SORT length(rows) DESC
```

## Сложность задач по разделам

```dataview
TABLE length(rows) AS "Задач"
FROM "Задачи"
GROUP BY раздел + " / " + сложность
SORT раздел, сложность
```
