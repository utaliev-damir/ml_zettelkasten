
# Структура знаний

Это рабочий vault для подготовки к ШАД и изучения Machine Learning. Здесь собраны 9 типов сущностей, каждая лежит в своей папке и имеет свой шаблон в `_templates/`.

## Типы заметок

| Тип               | Папка                       | Что это                                                                |
| ----------------- | --------------------------- | ---------------------------------------------------------------------- |
| Определение       | `Определения/`              | Точное мат. определение объекта                                        |
| Свойство          | `Свойства/`                 | Утверждение об объекте, обычно с доказательством                       |
| Теорема           | `Теоремы/`                  | Крупное утверждение (часто с собственным именем)                       |
| Метод             | `Методы/`                   | Подход к решению **класса** задач                                      |
| Алгоритм          | `Алгоритмы/`                | Конкретная процедура с псевдокодом и сложностью                        |
| Сигнал            | `Сигналы/`                  | Признак условия, указывающий на метод                                  |
| Структура данных  | `Структуры и типы данных/`  | Тип данных и операции над ним                                          |
| Теория алгоритмов | `Теория алгоритмов/`        | Теоретические понятия (асимптотика, сложность)                         |
| Задача            | `Задачи/`                   | Решённая задача с разбором                                             |

## Поле «формат» у задачи

| Значение         | Когда использовать                                        |
| ---------------- | --------------------------------------------------------- |
| `доказательство` | Доказать, проверить, обосновать                           |
| `вывод`          | Получить формулу или оценку - например, вывод производной |
| `численная`      | Посчитать конкретное значение                             |
| `код`            | Реализовать алгоритм, написать функцию                    |
