Алгоритм которые делит задачу на подзадачи, решает независимо подзадачи и возвращает результат в основную задачу. Важный этап - то как вернуть результаты подзадачи. 

## Сортировка

### SelectionSort
Пример сортировка выбором списка из n элементов - SelectionSort

```
SelectionSort


Проходим по списку:
	Ищем наименьший элемент
	Переставляем его на текущее место
	Текущий элемент переставляем на место наименьшего

Проделываем эту операцию уменьшая список
```

Сложность такого алгоритма $O(n^2)$ , так как всего операций мы сделаем $S = n + (n-1) + (n-2) + (n-3) +...+1$. А это $\frac{n(n-1)}{2}$ операций (по [[Прогрессия|арифметической прогрессии]])

### MergeSort

Но есть алгоритм, который выполняет сортировку гораздо быстрее. Это алгоритм MergeSort. Его основная идея заключается в том, что мы создаем отдельно функцию Merge, которая принимает на вход два отсортированных списка и объединяет их в один отсортированный список. Затем мы пишем функцию, которая будет бить входящий список на два и передавать их в Merge, но бить будем до тех пор пока не дойдем до единичных элементов. Вы итоге первая работа merge будет тогда, когда в него передадутся единичные элементы

```
MergeSort

Merge(List1, List2):
	while List1 and List2 are not empty:
		if first element from List1 < first element from List2:
			move first element from List1 to Sortedlist
		else:
			move first element from List2 to Sortedlist
	move rest of elements from List1 or List2 to SortedList
	return SortedList

MergeSort(List):
	first_half = first half of list
	second_half = second half of list
	SortedFirsthalf = MergeSort(first_half)
	SortedSecondhalf = MergeSort(second_half)
	SortedList = Merge(SortedFirsthalf, SortedSecondhalf)
	return SortedList
	
```

Сложность алгоритма равна $O(\log_{2}{n})$ - почему так, на каждом $i-ом$уровне рекурсии мы будем иметь $2^i$ списков, сначала 2, потом 4, 8 16и так далее пока списки не станут одноэлементными, на последнем уровне $2^k = n$ , степень k это количество уровней, то есть всего уровней будет $\log_{2}n$ , и на каждом уровне будет выполняться n действий.
