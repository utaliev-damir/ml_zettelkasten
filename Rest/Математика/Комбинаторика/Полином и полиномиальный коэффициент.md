
## __Полином это выражение типа__

$$(x_{1}+x_{2}+...+x_{n})^n$$
Я уже знаком с [[Бином Ньютона|бином Ньютона]], полином это обобщение над биномом, или бином это частный случай полинома. 

Возьмем для примера 

$(x_{1}+x_{2}+x{3})^2 = (x_{1}+x_{2}+x{3})(x_{1}+x_{2}+x{3}) = x_{1}^2 +x_{1}x_{2} +x_{1}x_{3}+x_{2}x_{1}+x_{2}^2+x_{2}x_{3}+x_{3}x_{1}+x_{3}x_{2}+x_{3}^2$
$=x_{1}^2+x_{2}^2+x_{3}^2 +2x_{1}x_{2}+2x_{1}x_{3}+2x_{2}x_{3}$

При каждом члене стоит коэффициент  $P(x_1,x_2,x_3)$, где 
$x_1$- количество повторов $x_1$ ,
$x_2$- количество повторов $x_2$, 
$x_3$- количество повторов $x_3$.

Мы умножаем 

$x_1, x_1 - \ одна \ комбинация, x_1, x_2 - \ две \ комбинации (x_1, x_2 \ и  \ x_2, x_1)$

То есть в результате умножения у нас получаются слагаемые состоящие из $n$ элементов. И результат будет равен сумме ведь этих слагаемых. Однако слагаемых может быть очень много и они могут быть идентичны по составу. Тогда чтобы узнать кол-во комбинаций слагаемых каждого типа можно применить полиномиальный коэффициент  $P(x_1,x_2,x_3)$ - при этом всего два элемента будут учтены (так одно слагаемое в нашем случае состоит из двух элементов.


## __Полиномиальная теорема в общем случае__ 

$(x_1+...+x_k)^n =$ сумме по всем возможным наборам чисел $n_1,...,n_k: ∀ n_i>=0 ; n_1+...+n_k=n$ выражений $P(n_1,...,n_k)x_1^{n_1}x_2^{n_2}...x_k^{n_k}$

где $n_k$ это степень при $k$-ом $x$, показывающая сколько раз этот $x$ встречается в выражении.

$$\sum_{(n_1,...,n_k);∀ n_i>=0 ; n_1+...+n_k=n}^{}P(n_1,...,n_k)x_1^{n_1}x_2^{n_2}...x_k^{n_k}$$
То есть слагаемых будет столько сколько существует способов разложить $n$ на сумму из $k$ неотрицательных слагаемых. Например $n$=3, $k$=3, тогда $(n_1,...,n_k)=(3,0,0), (0,3,0), (0,0,3), (1,1,1), (1,2,0), (1,0,2), (2,0,1)$и так далее. 
 
## __Доказательство полиномиальной теоремы__

Пусть есть выражение $(x_1+...+x_k)^n=(x_1+...+x_k)(x_1+...+x_k)...(x_1+...+x_k)$, тогда в результате перемножения у нас будут образовываться последовательности, где каждый элемент это элемент взятый из каждой скобки. Тогда пусть $n_1$ - это кол-во элементов $x_1$, $n_2$ - это кол-во элементов $x_2$, $n_k$ - это кол-во элементов $x_k$ в этой последовательности. При этом $n_i\ge0$, $n_1+...+n_k=n$.  Тогда такую последовательность можно представить в виде $x_1^{n_1}x_2^{n_2}...x_k^{n_k}$ . Но сколько таких последовательностей будет встречаться после умножения? Их будет столько, сколькими способами мы можем извлечь из $n$ скобок различных типов $x_k$ в $n_k$ количестве. Ну например у нас $k=3$, $n=3$ это значит что у нас есть три скобки и три типа икса. И у нас среди слагаемых может быть случай $x_1^2, x_2^1, x_3^0$. То есть у нас есть три скобки, и каждый тип $x$ может быть взят из этих скобок $n_k$ раз, и кол-во таких способов это как раз $P(n_1, n_2,..., n_k) = \frac{n!}{n_{1}!n_{2}!n_{3}! ... n_{k}!}$
