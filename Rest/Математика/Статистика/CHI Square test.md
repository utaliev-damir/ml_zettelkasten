Тест с помощью, которого можно проверить вероятность, что какая то выборка имеет определённое распределение (нормальное, дискретное Пуасона и т.д.)

Процесс расчета

1) Бьем выборку на бины
2) Считаем кол-во случаев в каждом бине
3) Считаем процентное распределение
4) Формируем нормальное распределение для заданного диапазона
5) Сравниваем два распределения с помощью p value
