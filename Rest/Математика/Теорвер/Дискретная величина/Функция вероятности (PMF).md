
Probability mass function - функция [[Аксиоматическое определение вероятности|вероятности]]. Способ определить чем равна вероятность случайной величины, с ее помощью можно определить распределение случайной величины. 

Функция вероятности  [[Дискретная случайная величина|дискретной случайной]] величины это такая функция $p_X$ что  $p_X (x)=P(X=x)$, при этом она положительная если $x \in \Omega$ или в [[Дискретная случайная величина#^527213|носителе]], в противном случае равна $0$.

PMF удовлетворяет следующим критериям:

1. $p_X(x) = 0 \ ; \forall x\notin \Omega$
2. $\sum_{j=1}^{n|\inf}p_X(x_j)=1$

| [[Дискретная случайная величина#^5bceff\|Без PMF]] ($P(X=x)$) | С PMF $p_X(x)$                                           |
| ------------------------------------------------------------- | -------------------------------------------------------- |
| Есть отдельные вероятности, но нет общей структуры.           | Есть функция, описывающая всё распределение.             |
| Сложно обобщать и писать формулы.                             | Можно компактно выражать вероятности и ожидания.         |
| Каждое событие нужно описывать вручную.                       | Можно считать всё автоматически по правилу суммирования. |

🔸 Вероятность $P(X=x)$ — это частный случай, «точечная информация».  
🔸 $p_X(x)$ — это полное **описание распределения** случайной величины.  
🔸 Зная PMF, ты можешь найти любую вероятность, ожидание, дисперсию и т.д.  
🔸 Без PMF — у тебя есть лишь отдельные фрагменты, но не вся картина.
