
## Определение
Градиентный спуск - алгоритм, позволяющий найти минимум функции. 

$$x = x - \alpha \frac{\delta}{\delta w}f(x)$$
В машинном обучении используется для поиска минимума функции потерь
$$w = w - \alpha \frac{\delta}{\delta w}J(w)$$
$$b = b - \alpha \frac{\delta}{\delta b}J(b)$$
где $J(x)$ - это функция потерь
$\alpha$ - это шаг обучения.

Что происходит при градиентном спуске - мы инициализируем некоторые веса и начинаем производить итерацию - из заданных значений вычитать значение производной в этой точке умноженное на шаг обучения.

## Градиентный спуск линейной регрессии

Для градиентного спуска нам понадобятся производные функций потерь от w и x

Модель
$$f(x) = wx +b$$
Функция потерь
$$J(x)=\frac{1}{2m}\sum_{i=1}^{m}((wx+b) - y)^2$$

Градиентный спуск пока не выполнится условие - значение функции близко к нулю или с каждой новой итерацией значение не сильно изменяется


$$w = w - \alpha \frac{\delta}{\delta w}(\frac{1}{2m}\sum_{i=1}^{m}((wx+b) - y)^2 = w - \alpha (\frac{1}{2m}\sum_{i=1}^{m}((wx+b) - y)^2*w $$
