Машинное обучение - это направление в науке, часть области искусственного интеллекта, изучающая алгоритмы, выявляющие закономерности в данных, и с помощью этих закономерностей может решать задачи классификации. регрессии, ранжирования, кластеризации. 
## Постановка задачи

Машинное обучение может быть двух видов:
* обучение с учителем
* обучение без учителя

Обучение с учителем - это задачи с размеченными данными, когда мы зачем таргет. Обучаем модель на размеченных данных, затем даем ей неразмеченный объект и на его признаках предсказываем таргет.

Обучение без учителя - это задачи без размеченных данных, у нас есть например только датасет объектов без таргета, с помощью такого подхода мы можем рассмотреть структуру данных, выделить группу объектов отличающихся друг от друг - кластеризация.

Типы задач определяются тем каким может быть множество ответов Y

Обучение с учителем:
1) $Y = R$  - задачи регрессии, когда нужно предсказать[[Множества#^0e06ce| вещественное число]], например цену квартиры, или прогноз погоды
2) $Y = \{1, 0\}$ - бинарная классификация
3) $Y = \{1, 2, 3, K\}$ - классификация - определяем принадлежность объекта к классу
4) $Y = \{1, 0\}^K$ - многомерная классификация, здесь для k класса определяется есть ли он (1) или его нет (0) у объекта
5) Y = (1, 2, 3, 4) - ранжирование, когда множество возможных объектов это упорядоченное множество

Обучение без учителя:
1) кластеризации
2) выявления аномалий
3)  сокращение размерности данных


## Критерии качества

Метрики - способ численно измерить то насколько модель нам помогает в жизни.
При решении задач машинного обучения мы можем работать с иерархией метрик. 
1) Бизнес метрики - например доходность сервиса
2) Онлайн метрики - например время сессии пользователя
3) Офлайн метрики - например отклонение предсказания от таргета

В классификации есть базовые метрики
1) accuracy - точность (например доля правильных диагнозов от общего кол-ва)
2) precision - например доля действительно больных от всех кому поставили такой диагноз
3) recall - полнота (например доля правильно определённых зараженных)

## Данные

Машинное обучение начинается с данных. В случае нехватки данных:
1) Краудсорсинг - привлечение людей готовых разметить данные. 
2) Обучение на неразмеченных данных - self supervised learning

Типы признаков
1) Числовые
2) Категориальные
3) Бинарные
4) Ординальные

При работе с данными могут встречаться такие проблемы
1) выбросы
2) пропуски
3) data-drift   - возможное изменение структуры данных со временем при использовании их в модели для обучения.

## Модель и алгоритм обучения

Обучающая выборка - это признаки и таргеты, данные на которых мы обучаем модель.

$x$ - признак
$y$ - целевая переменная
$(x,\ y)$ - один объект обучающей выборки
$(x_{i}, \ y_{i})$ - $i^{th}$ объект обучающей выборки
$m$ - количество объектов

Алгоритм машинного обучения это функция которая на вход принимает признак **x** и возвращает предсказание $\hat{y}$

$\hat{y}$ - предсказание, значение которое мы будем сравнивать с $y$

например [[Линейная регрессия в ML|линейная регрессия]]
$$f_{w,b}(x_{i}) = wx_{i}+b$$ Для оценки работы алгоритма или точности функции используют [[Функция потерь - Cost function|функции потерь]].
