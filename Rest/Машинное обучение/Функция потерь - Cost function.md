
После того как мы для $i-ого$ объекта сделали предсказание, нужно решить как его оценить. Для этого нам нужно сравнить целевую переменную с предсказанием.

# Регрессия

## MSE 

Средняя квадратичная ошибка
$$J(w,b) = \frac{1}{2m}  \sum^m_{i} (\hat{y_{i}} - y_{i})^2$$
или
$$J(w,b) = \frac{1}{2m}  \sum^m_{i} (f(x_{i}) - y_{i})^2$$

делим на 2 - для более легкого расчета градиентного спуска в дальнейшем.


> [!NOTE] Минусы
>- чувствительна к выбросам, представь, что в таргете есть выброс. Таргет: 1,2,100; Предсказание: 1,2,3. Тогда модель при обучении будет стремиться завысить предсказания для 1, 2 (которые вроде правильно предсказаны), для того бы средний квадрат был минимальным.

## MAE

Средняя абсолютная ошибка
$$J(w,b) = \frac{1}{2m}  \sum^m_{i} |\hat{y_{i}} - y_{i}|$$
> [!NOTE] Минусы
>- не дифференцируема

## MAPE

Mean Absolute Percentage Error
$$ J(w,b) = \frac{100\%}{n} \sum_{i=1}^{n} \left| \frac{y_i - \hat{y}_i}{y_i} \right| $$

Показывает насколько близко мы подобрались к ответу, относительная величина ошибки.

## Accuracy 

Доля правильных ответов, количество правильных ответов от общего кол-ва ответов
$$\text{Accuracy} = \frac{TP + TN}{TP + TN + FP + FN}$$

> [!NOTE] Минусы
> 
>  - Если у нас несбалансированная выборка по классам, то мы можем неправильно интерпретировать результаты. Например у нас 1000 объектов из них positive 950, negative 50. Если мы научились всегда правильно предсказывать первый класс и всегда неверно предсказывать второй класс у нас accuracy все равно будет равна 95% , вроде высокий показатель, но он вообще не умеет предсказывать второй класс.
> 
> - Не дифференцируема



## Precision

Доля правильно предсказанных положительных ответов от всех ответов, предсказанных  как положительные. 
$$\text{Accuracy} = \frac{TP}{TP + FP}$$

> [!NOTE] Минусы
> - не учитываются отрицательные ответы, а это второй класс объектов, нам может быть важным отрицательный класс 



## Recall

Полнота, доля правильно предсказанных положительных ответом от ответов, правильно предсказанных положительных  и неправильно предсказанных положительных
$$\text{Accuracy} = \frac{TP}{TP + FN}$$
> [!NOTE] Минусы
> - не учитываются  ложно положительные (ложные тревоги)



## Визуализация функции потерь

Для того, чтобы понять как связаны функция модели и функция потерь проведем эксперимент. Представим у нас есть функция предсказания $f_{w}(x)=wx$ и функция потерь $J(w)=\frac{1}{2m}\sum^m_{i}(f(x_{i})-y_{i})^2$  или  $J(w)=\frac{1}{2m}\sum^m_{i}(wx_{i}-y_{i})^2$ 
У нас есть три объекта, на которых можно обучиться - их таргеты и признаки равны 1,2,3 соответственно. Построим функцию потерь при разных w - и посмотрим при каких значениях она будет минимальная, параметром b пренебрежем.


Задача минимизировать функцию потерь
$$minimize \ J(w)$$


```python

import pandas as pd  
import seaborn as sns  
from matplotlib import pyplot as plt  
  
m = 3  
x = y = [1, 2, 3]  
  
def J(w):  
    res = 0  
    for i in range(m):  
        res += (w*x[i] - y[i])**2  
    return res/2*m  
  
results = []  
  
for w in range(-5, 8):  
    print('w:', w, 'J:', J(w))  
    results.append({'w': w, 'J': J(w)})  
  
df = pd.DataFrame(results)  
sns.lineplot(df, x='w', y='J')  
plt.show()
```
