**Итерация - процесс перебора элементов.**

**Итерируемый объект - объект, поддерживающий итерацию.**

К ним относятся  списки, кортежи, строки, множества, словари. То есть мы можем пройтись по этим объектам в итерации. По числам мы не можем так сделать. Как же понять, является ли объект итерируемым объектом? Для этого есть функция `iter()`. Благодаря этой функции мы можем во-первых понять является ли объект итерируемым или нет, а во-вторых можем превратить итерируемый объект в итератор. 

**Итератор - это объект-перечислитель.**  Это особый объект, который при каждом вызове отдает очередной элемент из итерируемого объекта. Когда итерация заканчивается, он выдает сообщение "StopIteration"

На самом деле мы регулярно сталкиваемся с итераторами - функции `map()`, `zip()`, `filter()` это функции которые возвращают итераторы.

Для вызова очередного элемента мы используем функцию `next()`

Даже в цикле, когда мы проходимся по итерируемому объекту, Python под капотом превращает его в итератор.

```python
for i in [1,2,3]:
	print(i)

# аналог
iterator = iter([1,2,3])
while True:
	try:
		print(next(iterator))
	except StopIteration:
		break
```

> Важно! По итератору можно пройтись один раз, потом он "опустошается" 

Преимущества итератора:
* Занимает меньше памяти, чем итерируемый объект.
* Хранит состояние вызовов.
