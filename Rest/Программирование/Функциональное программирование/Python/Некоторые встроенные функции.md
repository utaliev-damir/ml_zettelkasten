
## any(), all()

Обе функции принимают [[Итераторы|итерируемую последовательность]]. Каждый элемент последовательности можно представить как результат работы функции bool(), где такие элементы как 0, пустые коллекции, пустая строка, False, None будут возвращать False. А значения отличные от нуля, непустые коллекции, непустые строки, True будут возвращать True.

all() - возвращает True, если все элементы последовательности True.
any() - возвращает False, если все элементы последовательности False.

Можно эти функции так записать:
```python

def all(iterable):
	for element in iterable:
		if not element:
			return False
	return True

def any(iterable):
	for element in iterable:
		if element:
			return True
	return False
```

Поэтому при вызове `any([])` будет False, а `all([]`) будет True.
