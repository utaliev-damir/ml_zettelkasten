Генератор - это итератор, который можно обойти один раз
Итератор - это итерируемый объект, который можно использовать в функции next, которая запоминает какой следующий элемент должен быть вызван
Итерируемый объект - это объект, по элементам которого можно пройти в цикле.

Преимущества
* Экономия памяти - в python создать такой объект будет невозможно ``` [i for in range(1000000000000)]```, потому что он займет очень много памяти, а вот создать объект который не будет занимать столько памяти, но при этом мы сможем по нему итерироваться создать можно - это генератор

Ограничения
* Возможен один проход итерации - так как генератор запоминает только следующий элемент к вызову, когда мы по нему проходимся, то дойдя до последнего элемента, он не знает куда двигаться дальше и останавливается.
* Мы не можем обращаться к элементам итератора.
* Нельзя применять функции reversed, len

Пример кода
```python

a = [1, 2, 3]  # итерируемый объект

b = iter(a)  # итератор

c = (i for i in a)  # генератор

print(next(b))  
print(next(c))
```
