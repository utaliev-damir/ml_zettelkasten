#python

Замыкания в python это ситуация когда внутри функции объявляется другая функция, которая использует переменные из внешней функции, из[[Область видимости| области видимости]] nonlocal. Тогда при вызове outer x сохранится в inner

```python
def outer():
	x = 1
	def inner(y):
		return x + y
		
	return inner
```
