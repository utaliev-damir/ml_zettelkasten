#python 

Область видимости определяет часть программы в которой переменная видна, и с ней программа в этом месте может работать, и от каких частей переменная скрыты.

Из внутренней области видны внешние переменные, но не наоборот.

В python есть несколько областей видимости 

global - глобальная, переменные определены основной части программы

local - локальная, переменные определены внутри объектов, скрыты от глобальной зоны видимости

nonlocal - нелокальная, промежуточная область, обычно используется во вложенных функциях, чтобы внутренняя функция могла менять значение переменной из внешней. Может [[Замыкания|использоваться при замыкании]]

Одноименные ключевые слова используются при обращении из внутренней области видимости к внешней для изменения значения переменных.
