## Задача

Алгоритм решает задачу поиска мажоритарного элемента - элемента, который встречается больше чем $\lfloor \frac{n}{2} \rfloor$ раз, где $n$ - количество элементов массива.

## Идея

Алгоритм использует главное условие - мажоритарный элемент встречается больше, чем в половине случаев. 

Сначала выберем кандидата на мажоритарный элемент - пусть это будет первый элемент массива, и также создаем счетчик, если очередной элемент равен текущему кандидату - увеличиваем счетчик на 1, если не соответствует - уменьшаем счетчик на 1. Если счетчик доходит до нуля, обновляем кандидата на текущий и счетчик равен 1. 

Затем проверяем сколько раз наш кандидат встречался в исходном списке. Если больше, чем в половине случаев, значит это мажоритарный элемент.

## Когда применять

- поиск элемента, который встречается чаще  и при этом его больше чем в половине случаев

def find_majority_element(nums):

    major = nums[0]
    counter = 0
    
    for n in nums:
        
        
        if n == major:
            counter += 1
        else:
            counter -= 1
            
        if counter == 0:
            major = n
            counter = 1
    
    m_counter = 0
    
    for n in nums:
        if n == major:
            m_counter += 1
    
    if m_counter > math.floor(len(nums)/2):
        return major
    
    return -1
