## Задача

Поиск минимального или максимального значения в списке.

## Идея

Выбираем кандидата на значение мин/макс, каждый элемент последовательности сравниваем с кандидатом, если он больше/меньше делаем его кандидатом.

## Структура алгоритма

## Псевдокод

```python

def find_min(data):

	min_value = data[0] 
	
	for elem in data: 
		if elem < min_value: 
			min_value = elem 
			
	return min_value
```
