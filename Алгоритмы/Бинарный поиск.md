## Задача

Поиск элемента.

## Идея

Разделяй и властвуй. Находим середину и проверяем слева или справа от нее находится искомый элемент. Повторяем так пока не найдем элемент.

## Когда применять

Признаки задачи, при которых алгоритм подходит.

## Псевдокод

```python
def binary_search(data, target):
	left, right = 0, len(data)-1
	
	while left <= target:
		middle = (left + right)//2
		element = data[middle]
		if target == element:
			return middle
		elif target > element:
			left = middle + 1
		else:
			right = middle - 1
	return -1 
		
		
```

## Разбор корректности

Почему алгоритм действительно даёт правильный ответ.
