## Задача

Алгоритм решает задачу поиска путем последовательного перебора элементов.

## Идея

Сверяем очередной элемент с таргетным значением, в случае соответствия возвращаем True или индекс значения.

## Структура алгоритма

## Псевдокод

```python

def linear_search(nums, target):
	for n in nums:
		if n == target:
			return True
	return False
```
