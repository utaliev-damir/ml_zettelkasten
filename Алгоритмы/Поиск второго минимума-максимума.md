## Задача

Поиск второго минимума или максимума последовательности. Второй минимум - второй элемент в отсортированной последовательности по не убыванию. Второй максимум - предпоследний элемент в такой последовательности.

## Идея

Задаем две переменные, например для задачи поиска минимума, - min_1, min_2. Проходим по элементам массива, и проверяем их - если элемент меньше min_1, то обновляем min_2=min_1, min_1=элемент. Если элемент больше min_1, но меньше min_2, то min_2=элемент.

## Когда применять

- нужно использовать n-ые минимумы/максимумы 

## Псевдокод

```python
def find_second_min(nums):
	
	min_1 = min_2 = float("inf")
	for n in nums:
		if n < min_1:
			min_2 = min_1
			min_1 = n
		elif n < min_2:
			min_2 = n
			
	return min_2
```


## Типичные ошибки

- в первом сравнении сначала переопределить min_1, а потом min_2, тогда min_2 станет равным n
