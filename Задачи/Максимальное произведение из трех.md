## **Условие**

Реализуйте функцию `max_product_of_three()`, которая принимает один аргумент:

- `nums` – список целых чисел $3≤len(nums)≤10^6$

Функция должна находить в списке `nums` такую тройку элементов, произведение которых максимально. Возвращаемое значение функции – произведение найденных элементов.

## **Сигналы**

Какие признаки в условии подсказывают метод.

- нужно использовать $n$-ые максимумы/минимумы.

## **Основной метод**

Использование алгоритма поиска второго, третьего минимума/максимума

## **Ключевая идея**

У нас есть список чисел, в нем могут быть отрицательные и положительные числа, как выбрать три числа, чтобы произведение было максимальным? Нужно рассмотреть произведение трех максимальных неотрицательных числа, либо произведение наименьших двух отрицательных  и максимального неотрицательного. Выберем максимальное из них.

## **План решения**

```python

def max_product_of_three(nums):
    
    max_1 = max_2 = max_3 = float("-inf")
    min_1 = min_2 = float("inf")
    
    
    for n in nums:
        
        if n > max_1:
            max_3 = max_2
            max_2 = max_1
            max_1 = n
        elif n > max_2:
            max_3 = max_2
            max_2 = n
        elif n > max_3:
            max_3 = n
            
        if n < min_1:
            min_2 = min_1
            min_1 = n
        elif n < min_2:
            min_2 = n

    return max(
        max_1 * max_2 * max_3,
        min_1 * min_2 * max_1,
    )


```
