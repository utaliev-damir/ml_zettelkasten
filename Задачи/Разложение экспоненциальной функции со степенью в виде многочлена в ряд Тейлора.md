# Разложение $f(x) = e^{2x - x^2}$ в ряд Тейлора до $x^5$

## Условие

Написать разложение функции $f(x) = e^{2x - x^2}$ в ряд Тейлора по целым неотрицательным степеням $x$ до члена с $x^5$ включительно.

## Сигналы

Признаки в условии, которые подсказывают метод:

- Функция представима как $e^{g(x)}$, где $g(x)$ — многочлен — а для $e^t$ есть готовый табличный ряд.
- $g(x) = 2x - x^2$ начинается с $x^1$ (нет свободного члена) — значит $g^k(x)$ начинается с $x^k$, и для нужной точности достаточно конечного числа степеней.
- Требуется фиксированная точность ($x^5$), а не общая формула $n$-го члена — то есть нужен именно обрыв ряда, а не формула Тейлора через $f^{(n)}(0)$.

## Основной метод

**Подстановка в известное разложение** $e^t = \sum_{k=0}^{\infty} \tfrac{t^k}{k!}$, где $t = 2x - x^2$, с обрезанием на каждом шаге всего, что выше $x^5$.

Альтернатива (для проверки): $e^{2x - x^2} = e^{2x} \cdot e^{-x^2}$ — перемножить два готовых ряда.

## Ключевая идея

Поскольку $t = 2x - x^2$ имеет младшую степень $x^1$, то $t^k$ даёт вклад начиная с $x^k$. Значит для точности $x^5$ достаточно взять только $t, t^2, t^3, t^4, t^5$ — член с $t^6$ начинается с $x^6$ и выбрасывается. Это превращает бесконечный ряд в конечную сумму.

## План решения

1. Записать $e^t = 1 + t + \tfrac{t^2}{2!} + \tfrac{t^3}{3!} + \tfrac{t^4}{4!} + \tfrac{t^5}{5!} + o(t^5)$.
2. Подставить $t = 2x - x^2$ и вычислить $t^2, t^3, t^4, t^5$, **на каждом шаге отбрасывая** члены степени выше $x^5$.
3. Подставить все степени в ряд $e^t$ с соответствующими коэффициентами $1/k!$.
4. Раскрыть скобки и собрать подобные при $x^0, x^1, \dots, x^5$.

Промежуточные степени (после обрезания):

- $t = 2x - x^2$
- $t^2 = 4x^2 - 4x^3 + x^4$
- $t^3 = 8x^3 - 12x^4 + 6x^5$
- $t^4 = 16x^4 - 32x^5$
- $t^5 = 32x^5$


$$e^t = 1 + (2x - x^2) + \tfrac{(4x^2 - 4x^3 + x^4)^2}{2!} + \tfrac{(8x^3 - 12x^4 + 6x^5)^3}{3!} + \tfrac{(16x^4 - 32x^5)^4}{4!} + \tfrac{( 32x^5)^5}{5!} + o( 32x^5)$$

Сборка коэффициентов - берем каждый $x^k$ и смотрим какие коэффициенты в каждом $t$:

- $x^2$: $-1 + 2 = 1$
- $x^3$: $-2 + \tfrac{4}{3} = -\tfrac{2}{3}$
- $x^4$: $\tfrac{1}{2} - 2 + \tfrac{2}{3} = -\tfrac{5}{6}$
- $x^5$: $1 - \tfrac{4}{3} + \tfrac{4}{15} = -\tfrac{1}{15}$

**Ответ:** $$e^{2x - x^2} = 1 + 2x + x^2 - \tfrac{2}{3}x^3 - \tfrac{5}{6}x^4 - \tfrac{1}{15}x^5 + o(x^5)$$

## Ошибки / тупики

- Соблазн считать $f^{(k)}(0)$ напрямую — для композиции $e^{g(x)}$ это очень громоздко (пять производных от экспоненты от многочлена с раскрытием формулы Фаа-ди-Бруно).
- Легко забыть обрезать $t^k$ на промежуточных шагах и потащить в финальную сборку лишние члены степеней $\geq 6$ — арифметика разбухает.
- Арифметика дробей: общий знаменатель для $\tfrac{1}{2}, -2, \tfrac{2}{3}$ — это $6$, для $1, -\tfrac{4}{3}, \tfrac{4}{15}$ — это $15$.

## Инсайт

**Подстановка в табличный ряд почти всегда быстрее, чем формула Тейлора через производные**, если внутренняя функция — многочлен или что-то простое. Особенно сильно это работает, когда внутренняя функция $g(x)$ имеет нулевой свободный член: тогда $g^k$ автоматически даёт минимальную степень $x^k$, и нужно сложить лишь конечное число степеней. Этот же приём работает для $\sin(g(x))$, $\cos(g(x))$, $\ln(1 + g(x))$, $(1 + g(x))^\alpha$.


## Связанные задачи

- $f(x) = e^{\sin x}$ — та же идея, но $t = \sin x$, тоже без свободного члена; сначала разложить $\sin x$, затем подставить.
- $f(x) = \ln(1 + x + x^2)$ — подставить $g = x + x^2$ в ряд $\ln(1 + t)$.
- $f(x) = \cos(x^2 - x)$ — подстановка в ряд косинуса.
- $f(x) = e^{2x} \cdot \cos(3x)$ — перемножение двух табличных рядов, контрольный метод для текущей задачи.
