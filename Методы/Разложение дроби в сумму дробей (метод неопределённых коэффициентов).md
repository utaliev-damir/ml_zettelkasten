# Разложение дроби в сумму дробей (метод неопределённых коэффициентов)

## Идея
Одну «сложную» дробь вида $\dfrac{P(k)}{Q(k)}$ раскладывают в **сумму нескольких простых
дробей** — по одной на каждый множитель знаменателя. Это удобно, когда нужно
посчитать сумму ряда: соседние слагаемые сокращаются, и остаётся всего пара членов
(это называют телескопической суммой).

## Что означают термины
* **Правильная дробь** — та, где степень числителя *меньше* степени знаменателя.
  Например, $\dfrac{1}{k(k+1)}$ правильная (наверху степень $0$, внизу $2$),
  а $\dfrac{k^2}{k+1}$ — нет. Метод работает только для правильных дробей.
* **Линейный множитель** — множитель первой степени, вида $ak+b$: например $k$,
  $k+1$, $2k-3$. У него график — прямая.
* **Неприводимый квадратичный множитель** — множитель $ak^2+bk+c$, который *нельзя*
  разложить на линейные с действительными числами (нет действительных корней).
  Признак: дискриминант $b^2-4ac<0$. Пример: $k^2+1$ (корней среди чисел нет).

## Сигналы, что нужен этот метод
* Знаменатель — произведение множителей: $k(k+1)(k+2)$, $(2k-1)(2k+1)$, $k(k+2)$.
* Дробь правильная ($\deg P < \deg Q$).
* Множители — сдвиги друг друга ($k,\ k+1,\ k+2$) → сумма будет телескопировать.

## Схема применения
1. Убедиться, что дробь правильная. Если нет — поделить «уголком» и выделить целую часть.
2. Разложить знаменатель на множители.
3. Написать шаблон разложения с буквами-коэффициентами:
   * линейный $(ak+b)$ даёт $\dfrac{A}{ak+b}$;
   * повторяющийся $(ak+b)^m$ даёт $\dfrac{A_1}{ak+b}+\dots+\dfrac{A_m}{(ak+b)^m}$;
   * неприводимый квадратичный даёт $\dfrac{Bk+C}{ak^2+bk+c}$.
4. Найти коэффициенты: домножить на знаменатель и подставить корни (или приравнять
   коэффициенты при степенях $k$).
5. Сгруппировать так, чтобы члены сокращались.

## Пример использования
Найдём сумму $\displaystyle\sum_{k=1}^{\infty}\frac{1}{k(k+1)}$.

**Шаг 1.** Дробь правильная (наверху $0$, внизу $2$) ✓

**Шаг 2.** Знаменатель уже разложен: множители $k$ и $k+1$.

**Шаг 3.** Шаблон:
$$\frac{1}{k(k+1)} = \frac{A}{k} + \frac{B}{k+1}.$$

**Шаг 4.** Домножаем обе части на $k(k+1)$:
$$1 = A(k+1) + Bk.$$
Подставляем удобные значения $k$:
* $k=0:\quad 1 = A\cdot 1 \Rightarrow A = 1;$
* $k=-1:\quad 1 = B\cdot(-1) \Rightarrow B = -1.$

Значит $\dfrac{1}{k(k+1)} = \dfrac{1}{k} - \dfrac{1}{k+1}.$

**Шаг 5.** Расписываем сумму и видим сокращения:
$$
\sum_{k=1}^{n}\left(\frac{1}{k}-\frac{1}{k+1}\right)
=\left(1-\tfrac12\right)+\left(\tfrac12-\tfrac13\right)+\dots+\left(\tfrac1n-\tfrac1{n+1}\right)
= 1 - \frac{1}{n+1}.
$$
Устремляя $n\to\infty$, получаем $\displaystyle\sum_{k=1}^{\infty}\frac{1}{k(k+1)} = 1.$

## Типичные ошибки
* Забыли проверить правильность дроби ($\deg P \ge \deg Q$) → потеряли целую часть.
* Для повторяющегося множителя $(x-a)^2$ написали один член $\dfrac{A}{x-a}$
  вместо $\dfrac{A}{x-a}+\dfrac{B}{(x-a)^2}$.
* Для неприводимого квадратичного поставили константу $A$ вместо линейного $Bk+C$.
* Ошиблись в коэффициентах и не сделали проверку обратной подстановкой.
* При телескопировании неверно выписали оставшиеся члены или границы суммы.
