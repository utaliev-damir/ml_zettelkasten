## Общие формулы

|Название|Формула|
|---|---|
|Ряд Тейлора в точке $a$|$f(x) = \sum\limits_{n=0}^{\infty} \dfrac{f^{(n)}(a)}{n!}(x-a)^n$|
|Ряд Маклорена ($a = 0$)|$f(x) = \sum\limits_{n=0}^{\infty} \dfrac{f^{(n)}(0)}{n!} x^n$|
|Многочлен Тейлора $n$-й степени|$P_n(x) = \sum\limits_{k=0}^{n} \dfrac{f^{(k)}(a)}{k!}(x-a)^k$|
|Остаточный член (Лагранж)|$R_n(x) = \dfrac{f^{(n+1)}(\xi)}{(n+1)!}(x-a)^{n+1},\quad \xi \in (a,x)$|

> Все ряды ниже — разложения в точке $a = 0$ (Маклорена). Столбец «Сходимость» указывает, при каких $x$ ряд сходится к функции.

## Показательная функция

|Функция|Ряд|Сходимость|
|---|---|---|
|$e^x$|$\displaystyle\sum_{n=0}^{\infty} \dfrac{x^n}{n!} = 1 + x + \dfrac{x^2}{2!} + \dfrac{x^3}{3!} + \dots$|$x \in \mathbb{R}$|
|$a^x$|$\displaystyle\sum_{n=0}^{\infty} \dfrac{(\ln a)^n}{n!} x^n = 1 + x\ln a + \dfrac{(x\ln a)^2}{2!} + \dots$|$x \in \mathbb{R}$|

## Тригонометрические

| Функция  | Ряд                                                                                                                 | Сходимость               |
| -------- | ------------------------------------------------------------------------------------------------------------------- | ------------------------ |
| $\sin x$ | $\displaystyle\sum_{n=0}^{\infty} \dfrac{(-1)^n x^{2n+1}}{(2n+1)!} = x - \dfrac{x^3}{3!} + \dfrac{x^5}{5!} - \dots$ | $x \in \mathbb{R}$       |
| $\cos x$ | $\displaystyle\sum_{n=0}^{\infty} \dfrac{(-1)^n x^{2n}}{(2n)!} = 1 - \dfrac{x^2}{2!} + \dfrac{x^4}{4!} - \dots$     | $x \in \mathbb{R}$       |
| $\tan x$ | $x + \dfrac{x^3}{3} + \dfrac{2x^5}{15} + \dfrac{17x^7}{315} + \dots$                                                | $\|x\| < \dfrac{\pi}{2}$ |

## Логарифмические

|Функция|Ряд|Сходимость|
|---|---|---|
|$\ln(1+x)$|$\displaystyle\sum_{n=1}^{\infty} \dfrac{(-1)^{n+1} x^n}{n} = x - \dfrac{x^2}{2} + \dfrac{x^3}{3} - \dots$|$-1 < x \le 1$|
|$\ln(1-x)$|$\displaystyle -\sum_{n=1}^{\infty} \dfrac{x^n}{n} = -x - \dfrac{x^2}{2} - \dfrac{x^3}{3} - \dots$|$-1 \le x < 1$|
|$\ln\dfrac{1+x}{1-x}$|$\displaystyle 2\sum_{n=0}^{\infty} \dfrac{x^{2n+1}}{2n+1} = 2\left(x + \dfrac{x^3}{3} + \dfrac{x^5}{5} + \dots\right)$|$\|x\| < 1$|

## Степенные и биномиальный ряд

|Функция|Ряд|Сходимость|
|---|---|---|
|$\dfrac{1}{1-x}$|$\displaystyle\sum_{n=0}^{\infty} x^n = 1 + x + x^2 + x^3 + \dots$|$\|x\| < 1$|
|$\dfrac{1}{1+x}$|$\displaystyle\sum_{n=0}^{\infty} (-1)^n x^n = 1 - x + x^2 - x^3 + \dots$|$\|x\| < 1$|
|$\dfrac{1}{(1-x)^2}$|$\displaystyle\sum_{n=1}^{\infty} nx^{n-1} = 1 + 2x + 3x^2 + 4x^3 + \dots$|$\|x\| < 1$|
|$(1+x)^{\alpha}$|$\displaystyle\sum_{n=0}^{\infty} \binom{\alpha}{n} x^n = 1 + \alpha x + \dfrac{\alpha(\alpha-1)}{2!}x^2 + \dots$|$\|x\| < 1$|

где $\displaystyle\binom{\alpha}{n} = \dfrac{\alpha(\alpha-1)(\alpha-2)\cdots(\alpha-n+1)}{n!}$.

### Частные случаи биномиального ряда

|Функция|Ряд|Сходимость|
|---|---|---|
|$\sqrt{1+x}$|$1 + \dfrac{x}{2} - \dfrac{x^2}{8} + \dfrac{x^3}{16} - \dfrac{5x^4}{128} + \dots$|$\|x\| \le 1$|
|$\dfrac{1}{\sqrt{1+x}}$|$1 - \dfrac{x}{2} + \dfrac{3x^2}{8} - \dfrac{5x^3}{16} + \dots$|$\|x\| < 1$|

## Обратные тригонометрические

|Функция|Ряд|Сходимость|
|---|---|---|
|$\arctan x$|$\displaystyle\sum_{n=0}^{\infty} \dfrac{(-1)^n x^{2n+1}}{2n+1} = x - \dfrac{x^3}{3} + \dfrac{x^5}{5} - \dots$|$\|x\| \le 1$|
|$\arcsin x$|$\displaystyle\sum_{n=0}^{\infty} \dfrac{(2n)!}{4^n (n!)^2 (2n+1)} x^{2n+1} = x + \dfrac{x^3}{6} + \dfrac{3x^5}{40} + \dots$|$\|x\| \le 1$|

## Гиперболические

|Функция|Ряд|Сходимость|
|---|---|---|
|$\sinh x$|$\displaystyle\sum_{n=0}^{\infty} \dfrac{x^{2n+1}}{(2n+1)!} = x + \dfrac{x^3}{3!} + \dfrac{x^5}{5!} + \dots$|$x \in \mathbb{R}$|
|$\cosh x$|$\displaystyle\sum_{n=0}^{\infty} \dfrac{x^{2n}}{(2n)!} = 1 + \dfrac{x^2}{2!} + \dfrac{x^4}{4!} + \dots$|$x \in \mathbb{R}$|
|$\tanh x$|$x - \dfrac{x^3}{3} + \dfrac{2x^5}{15} - \dfrac{17x^7}{315} + \dots$|$\|x\| < \dfrac{\pi}{2}$|
|$\operatorname{artanh} x$|$\displaystyle\sum_{n=0}^{\infty} \dfrac{x^{2n+1}}{2n+1} = x + \dfrac{x^3}{3} + \dfrac{x^5}{5} + \dots$|$\|x\| < 1$|

## Шпаргалка: «голова» каждого ряда

|Функция|Первые члены|
|---|---|
|$e^x$|$1 + x + \dfrac{x^2}{2} + \dfrac{x^3}{6}$|
|$\sin x$|$x - \dfrac{x^3}{6} + \dfrac{x^5}{120}$|
|$\cos x$|$1 - \dfrac{x^2}{2} + \dfrac{x^4}{24}$|
|$\ln(1+x)$|$x - \dfrac{x^2}{2} + \dfrac{x^3}{3}$|
|$\dfrac{1}{1-x}$|$1 + x + x^2 + x^3$|
|$(1+x)^{\alpha}$|$1 + \alpha x + \dfrac{\alpha(\alpha-1)}{2}x^2$|

## Полезные приёмы

- **Замена переменной.** Чтобы получить ряд для $e^{-x^2}$, подставь $-x^2$ вместо $x$ в ряд $e^x$.
- **Дифференцирование/интегрирование.** Ряд можно почленно дифференцировать и интегрировать внутри интервала сходимости. Например, $\arctan x$ получается интегрированием ряда $\dfrac{1}{1+x^2}$.
- **Умножение и деление.** Ряды можно перемножать (свёртка коэффициентов) — так выводят, например, $\tan x = \dfrac{\sin x}{\cos x}$.
- **Сдвиг.** Для ряда в точке $a \ne 0$ удобно ввести $t = x - a$ и раскладывать по $t$.
