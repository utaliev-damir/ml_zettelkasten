# Таблица основных производных

## Базовые правила дифференцирования

|Правило|Формула|
|---|---|
|Производная константы|$(C)' = 0$|
|Производная аргумента|$(x)' = 1$|
|Вынесение константы|$(C \cdot f(x))' = C \cdot f'(x)$|
|Сумма / разность|$(f \pm g)' = f' \pm g'$|
|Произведение|$(f \cdot g)' = f' g + f g'$|
|Частное|$\left(\dfrac{f}{g}\right)' = \dfrac{f' g - f g'}{g^2}$|
|Сложная функция|$(f(g(x)))' = f'(g(x)) \cdot g'(x)$|
|Обратная функция|$(f^{-1})'(y) = \dfrac{1}{f'(x)}$|

## Степенные функции

|Функция|Производная|
|---|---|
|$x^n$|$n \cdot x^{n-1}$|
|$\sqrt{x}$|$\dfrac{1}{2\sqrt{x}}$|
|$\sqrt[n]{x}$|$\dfrac{1}{n \cdot \sqrt[n]{x^{n-1}}}$|
|$\dfrac{1}{x}$|$-\dfrac{1}{x^2}$|
|$\dfrac{1}{x^n}$|$-\dfrac{n}{x^{n+1}}$|

## Показательные и логарифмические

|Функция|Производная|
|---|---|
|$e^x$|$e^x$|
|$a^x$|$a^x \ln a$|
|$\ln x$|$\dfrac{1}{x}$|
|$\log_a x$|$\dfrac{1}{x \ln a}$|

## Тригонометрические

|Функция|Производная|
|---|---|
|$\sin x$|$\cos x$|
|$\cos x$|$-\sin x$|
|$\tan x$|$\dfrac{1}{\cos^2 x} = \sec^2 x$|
|$\cot x$|$-\dfrac{1}{\sin^2 x} = -\csc^2 x$|
|$\sec x$|$\sec x \cdot \tan x$|
|$\csc x$|$-\csc x \cdot \cot x$|

## Обратные тригонометрические

|Функция|Производная|
|---|---|
|$\arcsin x$|$\dfrac{1}{\sqrt{1 - x^2}}$|
|$\arccos x$|$-\dfrac{1}{\sqrt{1 - x^2}}$|
|$\arctan x$|$\dfrac{1}{1 + x^2}$|
|$\operatorname{arccot} x$|$-\dfrac{1}{1 + x^2}$|

## Гиперболические

|Функция|Производная|
|---|---|
|$\sinh x$|$\cosh x$|
|$\cosh x$|$\sinh x$|
|$\tanh x$|$\dfrac{1}{\cosh^2 x} = \operatorname{sech}^2 x$|
|$\coth x$|$-\dfrac{1}{\sinh^2 x}$|

## Обратные гиперболические

|Функция|Производная|
|---|---|
|$\operatorname{arsinh} x$|$\dfrac{1}{\sqrt{x^2 + 1}}$|
|$\operatorname{arcosh} x$|$\dfrac{1}{\sqrt{x^2 - 1}}$|
|$\operatorname{artanh} x$|$\dfrac{1}{1 - x^2}$|

## Часто встречающиеся сложные функции

|  Функция   | Производная            |
| ---------- | ---------------------- |
| $e^{kx}$   | $k \cdot e^{kx}$       |
| $a^{kx}$   | $k \cdot a^{kx} \ln a$ |
| $\ln(kx)$  | $\dfrac{1}{x}$         |
| $\ln$      | x                      |
| $\sin(kx)$ | $k \cos(kx)$           |
| $\cos(kx)$ | $-k \sin(kx)$          |
| $x^x$      | $x^x (\ln x + 1)$      |
