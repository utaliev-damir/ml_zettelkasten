## Формулировка


Векторное произведение векторов $\overline{a}\times\overline{b}$  - вектор, который $\perp \overline{a}$  и  $\perp \overline{b}$, его длина равна $|\overline{a}|\cdot|\overline{b}|\cdot \sin\alpha$, вектора $\overline{a}, \overline{b}, \overline{a}\times\overline{b}$ образуют правую тройку векторов $\overline{a}, \overline{b},\overline{c}$


## Свойства


1. $\overline{a}\times\overline{b}=-\overline{b}\times\overline{a}$
2. $(\alpha\overline{a})\times\overline{b}=\alpha\overline{a}\times\overline{b}$
3. $(\overline{a}+\overline{b})\times\overline{c}=\overline{a}\times\overline{c}+\overline{b}\times\overline{c}$

$$


Там где одинаковые орты, слагаемые будут равны 0.

$$=

## Дистрибутивность векторного произведения

Для любых векторов $\bar a,\bar b,\bar c$:

$$(\bar a + \bar b)\times\bar c = \bar a\times\bar c + \bar b\times\bar c.$$

### Доказательство

Чтобы доказать равенство двух векторов, умножим обе части скалярно на вектор $\bar b$ и сравним смешанные произведения.

**Шаг 1. Правая часть.**

$$\big(\bar a\times\bar c + \bar b\times\bar c\big)\cdot\bar b = (\bar a\times\bar c)\cdot\bar b + \underbrace{(\bar b\times\bar c)\cdot\bar b}_{=0} = (\bar a\times\bar c)\cdot\bar b.$$

Второе слагаемое равно нулю: вектор $\bar b\times\bar c$ перпендикулярен $\bar b$. Первое слагаемое в общем случае ненулевое.

**Шаг 2. Левая часть.**

$$\big[(\bar a + \bar b)\times\bar c\big]\cdot\bar b$$

— это смешанное произведение тройки $(\bar a+\bar b),\ \bar c,\ \bar b$, то есть (по модулю) **объём параллелепипеда**, построенного на этих трёх векторах.

**Шаг 3. Сравнение объёмов (рисунок).**

> Два параллелепипеда стоят на одном основании (параллелограмме на $\bar c$ и $\bar b$) и имеют одинаковую высоту: верхняя грань тела 1 получена из тела 2 сдвигом на $\bar b$ — вектор из плоскости основания. Поэтому их объёмы равны.
![[Pasted image 20260516184335.png]]


$$\operatorname{Vol}\big((\bar a+\bar b),\bar c,\bar b\big) = \operatorname{Vol}\big(\bar a,\bar c,\bar b\big).$$

Оба параллелепипеда построены на **общем основании** — параллелограмме на $\bar c$ и $\bar b$. Их третьи рёбра, $\bar a+\bar b$ и $\bar a$, отличаются на вектор $\bar b$, который **лежит в плоскости основания**. Сдвиг верхней грани вдоль вектора из плоскости основания не меняет высоту тела — меняется только его наклон. Площадь основания и высота одинаковы, значит

$$\operatorname{Vol}\big((\bar a+\bar b),,\bar c,,\bar b\big) = \operatorname{Vol}\big(\bar a,,\bar c,,\bar b\big).$$

**Вывод.** Скалярные произведения обеих частей на $\bar b$ совпали:

$$\big[(\bar a + \bar b)\times\bar c\big]\cdot\bar b = \big(\bar a\times\bar c + \bar b\times\bar c\big)\cdot\bar b.$$

Повторяя то же рассуждение для трёх некомпланарных направлений, получаем равенство самих векторов:

$$(\bar a + \bar b)\times\bar c = \bar a\times\bar c + \bar b\times\bar c. \qquad\blacksquare$$





## Эквивалентные формулировки

- для двух векторов  **p** × **q**  в двумерном пространстве - это скаляр, и его значение численно равно ориентированное площади параллелограмма натянутого на **p** и **q**. 
- для двух векторов  **p** × **q**  в трехмерном пространстве - это вектор, и его длина равна $|\overline{a}|\cdot|\overline{b}|\cdot \sin\alpha$, вектора $\overline{a}, \overline{b}, \overline{a}\times\overline{b}$ образуют правую тройку векторов $\overline{a}, \overline{b},\overline{c}$
- а объем параллелепипеда равен уже смешанному произведению
