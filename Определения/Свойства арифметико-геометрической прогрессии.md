## Формулировка

Арифметико-геометрическая прогрессия — последовательность, каждый член которой есть произведение соответствующих членов **арифметической** и **геометрической** прогрессий:

$$c_n = \bigl(a_1 + (n-1)d\bigr)\cdot q^{\,n-1}$$

Развёрнуто:
$$a_1,\;\;(a_1+d)q,\;\;(a_1+2d)q^2,\;\;(a_1+3d)q^3,\;\;\dots,\;\;(a_1+(n-1)d)q^{\,n-1}$$
Арифметическая часть $a_1+(n-1)d$ нарастает линейно, геометрическая $q^{\,n-1}$ умножается на $q$. Частные случаи: при $d=0$ — чистая геометрическая прогрессия, при $q=1$ — чистая арифметическая.

## Обозначения

* $a_1$ — первый член (он же первый член арифметической части)
* $d$ — разность арифметической части
* $q$ — знаменатель геометрической части

## Характеристическое свойство

У геометрической прогрессии каждый член — среднее геометрическое соседей. Здесь аналог — трёхчленное рекуррентное соотношение. Рассмотрим «геометрически поправленные разности» $g_n = c_{n+1} - q\,c_n$:

$$c_{n+1} - q\,c_n = \bigl(a_1+nd\bigr)q^{\,n} - \bigl(a_1+(n-1)d\bigr)q^{\,n} = d\,q^{\,n}.$$

То есть $g_n = d\,q^{\,n}$ — сами эти разности образуют геометрическую прогрессию со знаменателем $q$, значит $g_n = q\,g_{n-1}$. Раскрывая, получаем:

$$\boxed{\,c_{n+1} = 2q\,c_n - q^2\,c_{n-1}\,}$$

(характеристический многочлен $(x-q)^2$ имеет двойной корень $q$; общее решение $(A+Bn)q^n$).

## Свойства членов

Каждый следующий член выражается через предыдущий:

$$c_{n+1} = q\,c_n + d\,q^{\,n},\qquad c_1 = a_1.$$

## Сумма первых n членов

$$S_n = a_1 + (a_1+d)q + (a_1+2d)q^2 + \dots + (a_1+(n-1)d)q^{\,n-1}$$

Умножаем на $q$ и сдвигаем на одну позицию:

$$qS_n = a_1 q + (a_1+d)q^2 + \dots + (a_1+(n-2)d)q^{\,n-1} + (a_1+(n-1)d)q^{\,n}$$

Вычитаем. Ключ приёма: при вычитании столбцов одинаковых степеней арифметические коэффициенты $(a_1+kd)-(a_1+(k-1)d)$ гасятся до постоянного $d$:

$$S_n - qS_n = a_1 + d\bigl(q + q^2 + \dots + q^{\,n-1}\bigr) - (a_1+(n-1)d)q^{\,n}$$

Средняя скобка — обычная геометрическая прогрессия: $\;q+q^2+\dots+q^{\,n-1} = \dfrac{q - q^{\,n}}{1-q}.$ Тогда

$$S_n(1-q) = a_1 - (a_1+(n-1)d)q^{\,n} + \frac{d\,(q - q^{\,n})}{1-q}.$$

Делим на $(1-q)$:

$$\boxed{\,S_n = \frac{a_1 - \bigl(a_1+(n-1)d\bigr)q^{\,n}}{1-q} + \frac{d\,q\,\bigl(1 - q^{\,n-1}\bigr)}{(1-q)^2}\,}\qquad (q\neq 1)$$

**Проверка $d=0$:** $\;S_n = a_1\dfrac{1-q^{\,n}}{1-q}$ — обычная геометрическая сумма. ✓

**Числовая проверка** ($a_1=1,\ d=1,\ q=\tfrac12,\ n=3$): $c_n = n(\tfrac12)^{n-1}$, прямо $S_3 = 1+1+\tfrac34 = \tfrac{11}{4}$; формула даёт $\tfrac{11}{4}$. ✓

## Сумма бесконечной прогрессии

**Теорема.** Пусть дана арифметико-геометрическая прогрессия с параметрами $a_1,\,d,\,q$. При $|q|<1$

$$\sum_{n=1}^{\infty}\bigl(a_1+(n-1)d\bigr)q^{\,n-1} = \frac{a_1}{1-q} + \frac{d\,q}{(1-q)^2}.$$

**Вывод.**

1. **Конечная сумма.** Уже получена выше:
$$S_n = \frac{a_1 - \bigl(a_1+(n-1)d\bigr)q^{\,n}}{1-q} + \frac{d\,q\,\bigl(1 - q^{\,n-1}\bigr)}{(1-q)^2}.$$

2. **Предел.** Множители $a_1,\,d,\,q,\,(1-q)$ от $n$ не зависят; меняются только $q^{\,n}$ и $(a_1+(n-1)d)q^{\,n}$. При $|q|<1$:
   * $q^{\,n}\to 0$;
   * $(a_1+(n-1)d)\,q^{\,n}\to 0$ — экспонента $q^{\,n}$ убывает быстрее, чем линейно растёт множитель $a_1+(n-1)d$.

   Оба «хвостовых» слагаемых исчезают, и остаётся
$$S = \frac{a_1}{1-q} + \frac{d\,q}{(1-q)^2}. \qquad \blacksquare$$

Первое слагаемое $\dfrac{a_1}{1-q}$ — вклад геометрической части (как будто $d=0$), второе $\dfrac{dq}{(1-q)^2}$ — добавка от линейного нарастания $d$.

## Замечание о сходимости

Как и у геометрической прогрессии, сходимость определяется только знаменателем $q$ (параметры $a_1$ и $d$ на неё не влияют):

* $|q|<1$ — оба хвоста стремятся к нулю, ряд сходится к $\dfrac{a_1}{1-q}+\dfrac{dq}{(1-q)^2}$.
* $q=1$ — прогрессия становится арифметической, $c_n = a_1+(n-1)d$, частичные суммы растут → расходится.
* $|q|\ge 1$ (при $d\neq 0$) — модуль члена $|c_n|=|a_1+(n-1)d|\cdot|q|^{\,n-1}$ не стремится к нулю (даже растёт), необходимое условие сходимости нарушено → расходится.

**Итог:** ряд сходится тогда и только тогда, когда $-1<q<1$. Знак $q$ роли не играет — важен только модуль.

## Примеры

1. $a_1=1,\ d=2,\ q=3$: члены $1,\ 9,\ 45,\ 189,\ 729,\dots$ Так как $q>1$ — сумма расходится.

2. $a_1=1,\ d=1,\ q=\tfrac12$: тогда $c_n = n\left(\tfrac12\right)^{n-1}$, то есть $1,\ 1,\ \tfrac34,\ \tfrac12,\dots$
$$S = \frac{1}{1-\tfrac12} + \frac{1\cdot\tfrac12}{\left(1-\tfrac12\right)^2} = 2 + 2 = 4.$$

3. Классический ряд $\displaystyle\sum_{k=1}^{\infty}\frac{k}{2^{k}}$ — это случай $a_1=\tfrac12,\ d=\tfrac12,\ q=\tfrac12$ (так как $\tfrac{k}{2^k} = \left(\tfrac12+(k-1)\tfrac12\right)\left(\tfrac12\right)^{k-1}$):
$$S = \frac{\tfrac12}{1-\tfrac12} + \frac{\tfrac12\cdot\tfrac12}{\left(1-\tfrac12\right)^2} = 1 + 1 = 2.$$
