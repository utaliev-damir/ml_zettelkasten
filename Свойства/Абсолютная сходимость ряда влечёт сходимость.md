## Формулировка

Если ряд из модулей $\sum_{k=1}^{\infty} |a_k|$ сходится, то и сам ряд $\sum_{k=1}^{\infty} a_k$ сходится.

$$\sum_{k=1}^{\infty} |a_k| \text{ сходится} \quad\Longrightarrow\quad \sum_{k=1}^{\infty} a_k \text{ сходится}$$

## Условия

Предположения, при которых утверждение верно.

- Ряд из модулей $\sum_{k=1}^{\infty} |a_k|$ **сходится** (то есть ряд $\sum a_k$ сходится абсолютно).
- Члены $a_k$ — вещественные или комплексные числа. (Существенна полнота пространства: критерий Коши даёт сходимость только там, где всякая фундаментальная последовательность имеет предел.)

## Идея доказательства

Кусок исходного ряда — это **модуль суммы**, а кусок ряда из модулей — **сумма модулей**. По неравенству треугольника модуль суммы не превосходит суммы модулей. Значит кусок $\sum a_k$ зажат сверху куском $\sum |a_k|$, который уже мал (раз ряд модулей сходится). Поэтому и кусок $\sum a_k$ мал — выполнен критерий Коши.

## Доказательство

Ряд $\sum |a_k|$ сходится, поэтому для него выполнен критерий Коши. Так как все слагаемые неотрицательны, модуль их суммы равен самой сумме:

$$\forall \epsilon > 0 \quad \exists N(\epsilon): \quad \forall n > N(\epsilon),\quad \forall m \geq 1 \quad |a_{n+1}| + |a_{n+2}| + \dots + |a_{n+m}| < \epsilon$$

По неравенству треугольника (модуль суммы не больше суммы модулей):

$$|a_{n+1} + a_{n+2} + \dots + a_{n+m}| \quad\le\quad |a_{n+1}| + |a_{n+2}| + \dots + |a_{n+m}|$$

Соединяя, для тех же $n > N(\epsilon)$ и всех $m \geq 1$:

$$|a_{n+1} + a_{n+2} + \dots + a_{n+m}| \quad\le\quad |a_{n+1}| + \dots + |a_{n+m}| \quad<\quad \epsilon$$

Значит куски ряда $\sum a_k$ тоже меньше $\epsilon$, то есть выполнен критерий Коши уже для $\sum a_k$. Следовательно, ряд $\sum a_k$ сходится. $\blacksquare$

## Следствия

- **Удобный способ проверки сходимости.** Чтобы доказать сходимость знакопеременного ряда, достаточно показать сходимость ряда из модулей — а для рядов с неотрицательными членами есть много готовых признаков (сравнения, Даламбера, радикальный Коши, интегральный).
    
- **Обратное неверно.** Из сходимости $\sum a_k$ не следует сходимость $\sum |a_k|$. Пример: $\sum \dfrac{(-1)^k}{k}$ сходится, а $\sum \dfrac{1}{k}$ расходится. Такие ряды называют **условно сходящимися**.
    
- **Терминология.** Если $\sum |a_k|$ сходится, говорят, что ряд сходится **абсолютно**. Абсолютная сходимость — более сильное свойство, чем обычная (условная).
