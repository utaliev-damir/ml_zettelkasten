## Формулировка

$$\lim_{x\to a}(f(x)+g(x))=\lim_{x\to a}f(x)+\lim_{x\to a}g(x)$$

## Условия

Предположения, при которых утверждение верно.

- Существует конечный предел $\lim_{x\to a}f(x)=A$
- Существует конечный предел $\lim_{x\to a}g(x)=B$

## Идея доказательства

Используем определение предела по Коши. Поскольку $f(x)$ можно сделать сколь угодно близкой к $A$, а $g(x)$ — сколь угодно близкой к $B$, то их сумма $f(x)+g(x)$ будет сколь угодно близка к $A+B$. Отклонение суммы оцениваем через неравенство треугольника, разбивая $\varepsilon$ пополам для каждой функции.

## Доказательство

Пусть $\lim_{x\to a}f(x)=A$ и $\lim_{x\to a}g(x)=B$. Покажем, что $\lim_{x\to a}(f(x)+g(x))=A+B$.

Зафиксируем произвольное $\varepsilon>0$.

По определению предела для $f(x)$: для числа $\dfrac{\varepsilon}{2}>0$ существует $\delta_1>0$, такое что $$\forall x: 0<|x-a|<\delta_1 \Rightarrow |f(x)-A|<\frac{\varepsilon}{2}$$

По определению предела для $g(x)$: для числа $\dfrac{\varepsilon}{2}>0$ существует $\delta_2>0$, такое что $$\forall x: 0<|x-a|<\delta_2 \Rightarrow |g(x)-B|<\frac{\varepsilon}{2}$$

Положим $\delta=\min(\delta_1,\delta_2)$. Тогда для любого $x$, удовлетворяющего $0<|x-a|<\delta$, выполняются оба неравенства одновременно, и по неравенству треугольника:

$$|(f(x)+g(x))-(A+B)|=|(f(x)-A)+(g(x)-B)|\leq |f(x)-A|+|g(x)-B|<\frac{\varepsilon}{2}+\frac{\varepsilon}{2}=\varepsilon$$

Таким образом, по определению предела $\lim_{x\to a}(f(x)+g(x))=A+B$. $\blacksquare$

## Следствия

- Предел разности функций равен разности их пределов: $\lim_{x\to a}(f(x)-g(x))=\lim_{x\to a}f(x)-\lim_{x\to a}g(x)$
- По индукции свойство обобщается на любое конечное число слагаемых: $\lim_{x\to a}\sum_{i=1}^n f_i(x)=\sum_{i=1}^n \lim_{x\to a}f_i(x)$
- Постоянный множитель можно выносить за знак предела (как частный случай): $\lim_{x\to a}(c\cdot f(x))=c\cdot\lim_{x\to a}f(x)$
