## Формулировка  
  
$$h(x)=f(x)\cdot g(x),\exists f^\prime(x),\exists g^\prime(x)$$
$$
h^\prime(x)=f(x)g^\prime(x)+g(x)f^\prime(x)
$$


## Условия

$$\exists f^\prime(x),\exists g^\prime(x)$$
## Идея доказательства
  
- Раскрываем $h^\prime(x)$ через определение
- С помощью группировки членов выражения приходим к пределу суммы и потом к сумме пределов. 
- Находим чему равен каждый предел
- В одном пределе будет выражение $f(x+\Delta)$, чтобы и мы сможем применить свойство предела - предел композиций функций.

## Доказательство

$$
h^\prime(x)=
	\lim_{\Delta\to0}
	\frac
		{
			f(x+\Delta)\cdot g(x+\Delta)-f(x)\cdot g(x)
			+f(x+\Delta)\cdot g(x)-f(x+\Delta)\cdot g(x)
		}
		{\Delta}
	
$$
$$
h^\prime(x)=
	\lim_{\Delta\to0}
	\frac
		{
			f(x+\Delta)(g(x+\Delta)-g(x))+g(x)( (f(x)+\Delta)-f(x))
		}
		{\Delta}
	
$$
$$
h^\prime(x)=
	\lim_{\Delta\to0}
	\frac
		{
			f(x+\Delta)(g(x+\Delta)-g(x))
		}
		{\Delta}
		+
	\lim_{\Delta\to0}
	\frac
		{
			g(x)( (f(x)+\Delta)-f(x))
		}
		{\Delta}
	
$$
Так как $\exists f^\prime(x)$, значит $f(x)$ непрерывная

$$

	\lim_{\Delta\to0}f(x+\Delta)=f(\lim_{\Delta\to0}x+\Delta)=f(x)
	
$$
$$
h^\prime(x)=
	\lim_{\Delta\to0}
	\frac
		{
			f(x)(g(x+\Delta)-g(x))
		}
		{\Delta}
		+
	\lim_{\Delta\to0}
	\frac
		{
			g(x)( (f(x)+\Delta)-f(x))
		}
		{\Delta}
	
$$
$$
h^\prime(x)=f(x)g^\prime(x)+g(x)f^\prime(x)
$$
