## Характеристическое свойство

Любой член, кроме первого, равен среднему геометрическому двух рядом с ним стоящих членов. 

$$|b_n|=\sqrt{b_{n-1}b_{n+1}}$$

## Свойство знаменателя

$$q=\frac{b_{n+1}}{b_n}$$

## Свойства членов геометрической прогрессии

$$b_n=b_{n-1}q$$
## Сумма первых n членов

$$S_n=b_1+q\cdot b_1+q^2\cdot b_1+q^3\cdot b_1+...+q^{n-1}\cdot b_1$$
$$qS_n=qb_1+q^2\cdot b_1+q^3\cdot b_1+q^4\cdot b_1+...+q^{n}\cdot b_1$$
$$S_n-qS_n=b_1q^n-b_1$$
$$S_n(1-q)=b_1q^n-b_1$$
$$S_n=\frac{b_1q^n-b_1}{1-q}$$
## Сумма бесконечной геометрической прогрессии

**Теорема.** Пусть дана геометрическая прогрессия с первым членом $b_1$ и знаменателем $q$. При $|q|<1$
$$\sum_{k=0}^{\infty} b_1 q^k = b_1 + b_1 q + b_1 q^2 + \dots = \frac{b_1}{1-q}.$$

### Вывод

**1. Конечная сумма.** Для частичной суммы $S_N = b_1 + b_1 q + b_1 q^2 + \dots + b_1 q^N$ умножаем на $q$ и вычитаем:
$$S_N - qS_N = b_1 - b_1 q^{N+1}\;\Longrightarrow\; S_N(1-q) = b_1\bigl(1 - q^{N+1}\bigr).$$
При $q\ne 1$:
$$S_N = b_1\,\frac{1 - q^{N+1}}{1 - q}.$$

**2. Предел.** Сумма ряда — это $\lim\limits_{N\to\infty} S_N$. Множители $b_1$ и $\frac{1}{1-q}$ от $N$ не зависят, изменяется только $q^{N+1}$. Так как $|q|<1$, имеем $q^{N+1}\to 0$, поэтому
$$S_N \longrightarrow b_1\,\frac{1 - 0}{1 - q} = \frac{b_1}{1-q}. \qquad \blacksquare$$

### Частный случай

При $b_1 = 1$ получаем формулу, которую часто записывают отдельно:
$$\sum_{k=0}^{\infty} q^k = 1 + q + q^2 + \dots = \frac{1}{1-q}.$$
Общая формула получается из неё вынесением $b_1$ за скобку: $\;b_1(1+q+q^2+\dots) = \dfrac{b_1}{1-q}$.

### Замечание о сходимости

Сходимость зависит только от $q$ (множитель $b_1$ на неё не влияет):

- $|q|<1$ — степени $q^{N+1}\to 0$, ряд **сходится** к $\dfrac{b_1}{1-q}$.
- $q=1$ — все члены равны $b_1$, частичные суммы $S_N = (N+1)b_1$ неограниченно растут → **расходится**.
- $q=-1$ — частичные суммы скачут между $b_1$ и $0$, предела нет → **расходится**.
- $|q|>1$ — $q^{N+1}$ неограниченно растёт по модулю → **расходится**.

Итог: ряд сходится тогда и только тогда, когда $-1 < q < 1$. Знак $q$ значения не имеет — важен только модуль.

### Примеры

При $b_1 = 1,\ q=\tfrac12$:
$$1+\tfrac12+\tfrac14+\tfrac18+\dots=\frac{1}{1-\tfrac12}=2.$$

При $b_1 = 6,\ q=\tfrac12$:
$$6+3+\tfrac32+\tfrac34+\dots=\frac{6}{1-\tfrac12}=12.$$

При $b_1 = 1,\ q=-\tfrac12$ (отрицательный знаменатель, $|q|<1$):
$$1-\tfrac12+\tfrac14-\tfrac18+\dots=\frac{1}{1-(-\tfrac12)}=\frac{2}{3}.$$
