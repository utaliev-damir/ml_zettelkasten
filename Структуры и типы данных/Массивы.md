## Назначение

Хранит упорядоченные данные в памяти.

## Основная идея

Данные хранятся в виде упорядоченной последовательной коллекции, где у каждого элемента есть номер.

## Свойства

- Размер массива определен
- Хранит данные одного типа
- Каждому элементу выделено одинаковое кол-во памяти

## Поддерживаемые операции

- Индексация
- Добавление/удаление элементов
- Сортировка

## Массивы в Python

List и Tuple это реализация массивов в Python. Но они ведь могут хранить разные типы данных. Да, но в памяти компьютера будет храниться массив с целыми числами - ссылками на ячейки памяти, где хранятся объекты из списка, которые могут иметь разный тип.

А вот numpy.array хранит данные без ссылок, напрямую, поэтому он хранит только элементы одинакового типа и выполняет операции быстрее.

## Сложность операций

|Операция со списком|Временная сложность|Пространственная сложность|
|---|---|---|
|`list[index]`|$O(1)$|$O(1)$|
|`list[index] = value`|$O(n)$|$O(1)$|
|`del list[index]`|$O(n)$|$O(1)$|
|`list.append(value)`|$O(1)$|$O(1)$|
|`list.insert(index, value)`|$O(n)$|$O(1)$|
|`list.pop()`|$O(1)$|$O(1)$|
|`list.pop(index)`|$O(n)$|$O(1)$|
|`value in list`|$O(n)$|$O(1)$|
|`for value in list`|$O(n)$|$O(1)$|
|`list.sort()`|$O(nlogn)$|$O(n)$|
|`list.clear()`|$O(n)$|$O(1)$|
|`list.reverse()`|$O(n)$|$O(1)$|
|`list.extend(iterable)`|$O(k)$|$O(k)$|
|`list.copy()`|$O(n)$|$O(n)$|
|`list.count(value)`|$O(n)$|$O(1)$|
|`list.remove(value)`|$O(n)$|$O(1)$|
|`list.index(value)`|$O(n)$|$O(1)$|

В методах `insert, pop(index), index(value), remove(value)` сложность $O(n)$ - так как происходит смещение.

Для создания списков, где элементы это члены прогрессии, лучше использовать `range(start, stop, step)`, по которому можно индексироваться, итерироваться, а также использовать в функциях `sum, len, min, max`. Так как `range` вычисляется с помощью формул, как прогрессия.

## Когда использовать

Используется в различных задачах, где нужно хранить данные упорядоченно.
