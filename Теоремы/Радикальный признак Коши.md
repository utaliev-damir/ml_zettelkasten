## Формулировка

Пусть дан ряд $\sum a_n$ с положительными членами ($a_n > 0$) и существует предел $$\rho = \lim_{n\to\infty} \sqrt[n]{a_n}.$$ Тогда:

- при $\rho < 1$ ряд $\sum a_n$ **сходится**;
- при $\rho > 1$ ряд $\sum a_n$ **расходится**;
- при $\rho = 1$ признак ответа не даёт (нужны другие методы).

## Условия

- $a_n > 0$ для всех $n$ (ряд с положительными членами).
- Существует предел корня $n$-й степени $\rho = \lim\limits_{n\to\infty} \sqrt[n]{a_n}$.

## Интуиция

$\sqrt[n]{a_n}$ — это «средний геометрический темп» убывания членов. Если он в пределе меньше единицы, то $a_n$ ведёт себя как $r^n$ с $r < 1$, то есть как сходящаяся геометрическая прогрессия. Если же темп больше единицы, члены не стремятся к нулю — ряд расходится.

## Идея доказательства

Зажать предельный корень строго между $\rho$ и единицей числом $r$ ($\rho < r < 1$). Тогда с некоторого момента $\sqrt[n]{a_n} < r$, откуда сразу $a_n < r^n$ — члены ряда мажорируются геометрической прогрессией, к которой применим признак сравнения.

## План доказательства

1. Рассмотреть случай $\rho < 1$; выбрать $r$ так, что $\rho < r < 1$.
2. По определению предела показать, что с некоторого номера $N$ выполнено $\sqrt[n]{a_n} < r$.
3. Возвести в степень $n$ и получить оценку $a_n < r^n$ (мажорирование геометрической прогрессией).
4. Применить признак сравнения с $\sum r^n$ ($r < 1$ ⟹ сходится) и заключить о сходимости; случай $\rho > 1$ разобрать аналогично.

## Доказательство

**Случай $\rho < 1$ (сходимость).**

Пусть $\rho = \lim\limits_{n\to\infty} \sqrt[n]{a_n} = L < 1$. Выберем число $r$ так, что $$L < r < 1.$$

По определению предела, начиная с некоторого момента (с номера $N$), верно $$\sqrt[n]{a_n} < r \qquad \text{для всех } n \ge N.$$

Возводя обе части в степень $n$, получаем $$a_n < r^n \qquad (n \ge N).$$

Значит, члены ряда (начиная с $N$) мажорируются членами геометрической прогрессии $r^n$. Так как при $r < 1$ ряд $\sum r^n$ сходится, то по признаку сравнения сходится и $\sum a_n$.

**Случай $\rho > 1$ (расходимость).**

Аналогично: начиная с некоторого момента $\sqrt[n]{a_n} > 1$, то есть $a_n > 1$. Члены ряда не стремятся к нулю, поэтому необходимое условие сходимости $a_n \to 0$ нарушено — ряд расходится. $\blacksquare$

## Следствия

- Удобен для рядов, где $a_n$ содержит степень вида $n$-й (например, $\left(\frac{n}{n+1}\right)^{n^2}$, $\frac{1}{(\ln n)^n}$) — корень $n$-й степени красиво упрощается.
- Радикальный признак «сильнее» признака Даламбера: если последний даёт результат, то и Коши даёт тот же; но есть ряды, где Коши работает, а Даламбер нет.
- Через $\limsup\limits_{n\to\infty} \sqrt[n]{|a_n|}$ формулируется формула Коши–Адамара для радиуса сходимости степенного ряда.

## Где используется

- Исследование числовых рядов на сходимость (особенно с членами вида «что-то в степени $n$»).
- Нахождение радиуса сходимости степенных рядов (формула Коши–Адамара).
