## Формулировка

Если последовательность ограниченна сверху/снизу и монотонно возрастает/убывает, то у этой последовательности есть предел.

Рассмотрим случай когда последовательность возрастает и ограничена сверху, значит есть некий $M: a_n<M \ \forall n$. 

## Доказательство

Пусть $a_n$ — **монотонно возрастающая** и **ограниченная сверху** последовательность.  
Пусть $l_1 = a_1$ — нижняя граница, а $h_1 = M$ — любая верхняя граница последовательности.

**Идея:** будем строить последовательности $l_n$ и $h_n$, которые с каждым шагом «сужают интервал», в котором находится предел.

1. Задаём середину интервала:  
    $m_n = \frac{l_n + h_n}{2}$

2. Обновляем границы в зависимости от положения $a_n$:  
    $\text{если } a_n \ge m_n, \quad l_{n+1} = a_n, \ h_{n+1} = h_n$
    $\text{если } a_n < m_n, \quad l_{n+1} = l_n, \ h_{n+1} = m_n$

3. Получаем последовательности ($l_n \le a_n \le h_n$) с интервалом ($l_n, h_n$), который **монотонно сужается**:  
    $h_n - l_n \to 0 \quad \text{при } n \to \infty$

4. В пределе:  
    $\lim_{n \to \infty} l_n = \lim_{n \to \infty} h_n = L$
    и, так как ($l_n \le a_n \le h_n$) по [[Теорема о двух полицейских]], имеем:  
    $\lim_{n \to \infty} a_n = L$
## Строгий вариант

Пусть $a_n$ — возрастающая и ограниченная сверху последовательность.
Обозначим множество её значений:  
$$A = {a_n \mid n \in \mathbb{N}}.  $$
Так как $a_n$ ограничена сверху, существует верхняя граница $M$, то есть:  
$$a_n \le M \quad \forall n \in \mathbb{N}.  $$
По **аксиоме полноты** множества вещественных чисел множество (A) имеет **точную верхнюю грань** (супремум):  
$$L = \sup A$$
Докажем, что  
$$\lim_{n \to \infty} a_n = L.  $$
По определению супремума $L = \sup A$:
Для любого ($\varepsilon > 0$) существует ($N$), такое что  
$$L - \varepsilon < a_N \le L.  $$
Так как последовательность возрастает, то для всех ($n > N$):   $$L - \varepsilon < a_N \le a_n \le L. $$
Отсюда  
$$|a_n - L| < \varepsilon, \quad \forall n > N$$
По определению предела: 
$$\boxed{\lim_{n \to \infty} a_n = L}  $$
