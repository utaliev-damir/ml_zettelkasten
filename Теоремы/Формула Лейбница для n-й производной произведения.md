## Условие

Доказать, что для функций $f$ и $g$, дифференцируемых $n$ раз:

$$(f(x)g(x))^{(n)} = \sum_{k=0}^{n} C_n^k  g^{(k)}(x)  f^{(n-k)}(x)$$

## Сигналы

- В формуле — биномиальные коэффициенты $C_n^k$ и сумма по $k$ от 0 до $n$. Структура та же, что у бинома Ньютона.
- Утверждение для всех натуральных $n$ — значит, индукция.
- База $n=1$ — это обычное правило производной произведения, оно уже известно.
- n-ая производная произведения функций

## Метод

Индукция по $n$ + сдвиг индекса суммирования + тождество Паскаля.

## Идея

После шага индукции получаются две суммы. В лоб их сложить нельзя — в одной у $g$ стоит $g^{(k+1)}$, а в другой $g^{(k)}$. Если сдвинуть индекс в первой сумме, у обеих окажется одинаковая «структурная часть» $g^{(k)} f^{(n+1-k)}$. После этого слагаемые с одинаковыми $k$ складываются, а коэффициенты при них сворачиваются по Паскалю: $C_n^{k-1} + C_n^k = C_{n+1}^k$.

## План решения

1. База $n=1$: $(fg)' = f'g + fg'$ — это и есть формула при $n=1$.
2. Шаг индукции: дифференцирую обе части предположения, в каждом слагаемом применяю правило произведения.
3. Получаю две суммы $S_1$ и $S_2$.
4. В $S_1$ делаю сдвиг $k \to k-1$, чтобы выровнять структурную часть с $S_2$.
5. Складываю «столбиком» по одинаковым $k$. Краешки ($k=0$ и $k=n+1$) дают по одному слагаемому, серединка ($k=1,\dots,n$) — пары.
6. Применяю Паскаль и собираю всё в одну сумму.

### После дифференцирования

$$(fg)^{(n+1)} = \underbrace{\sum_{k=0}^{n} C_n^k , g^{(k+1)} f^{(n-k)}}_{S_1} + \underbrace{\sum_{k=0}^{n} C_n^k , g^{(k)} f^{(n+1-k)}}_{S_2}$$

Сдвиг индекса в $S_1$ ($k \to k-1$):

$$S_1 = \sum_{k=1}^{n+1} C_n^{k-1}  g^{(k)} f^{(n+1-k)}$$

## Сложение «столбиком» при $n = 4$

Цель — получить формулу для $n+1 = 5$.

**$S_2$** ($k = 0, \dots, 4$):

| $k$ | слагаемое                |
| --- | ------------------------ |
| 0   | $C_4^0  g^{(0)} f^{(5)}$ |
| 1   | $C_4^1 g^{(1)} f^{(4)}$  |
| 2   | $C_4^2  g^{(2)} f^{(3)}$ |
| 3   | $C_4^3  g^{(3)} f^{(2)}$ |
| 4   | $C_4^4  g^{(4)} f^{(1)}$ |

**$S_1$ после сдвига** ($k = 1, \dots, 5$):

| $k$ | слагаемое                |
| --- | ------------------------ |
| 1   | $C_4^0  g^{(1)} f^{(4)}$ |
| 2   | $C_4^1  g^{(2)} f^{(3)}$ |
| 3   | $C_4^2  g^{(3)} f^{(2)}$ |
| 4   | $C_4^3  g^{(4)} f^{(1)}$ |
| 5   | $C_4^4  g^{(5)} f^{(0)}$ |

Складываю по столбикам — структурная часть $g^{(k)} f^{(5-k)}$ одинаковая:

|$k$|из $S_2$|из $S_1$|сумма коэффициентов|по Паскалю|
|---|---|---|---|---|
|0|$C_4^0$|—|$C_4^0$|$= C_5^0$|
|1|$C_4^1$|$C_4^0$|$C_4^1 + C_4^0$|$= C_5^1$|
|2|$C_4^2$|$C_4^1$|$C_4^2 + C_4^1$|$= C_5^2$|
|3|$C_4^3$|$C_4^2$|$C_4^3 + C_4^2$|$= C_5^3$|
|4|$C_4^4$|$C_4^3$|$C_4^4 + C_4^3$|$= C_5^4$|
|5|—|$C_4^4$|$C_4^4$|$= C_5^5$|

Края ($k=0$ и $k=5$) — там слагаемое только в одной сумме, но $C_4^0 = C_5^0 = 1$ и $C_4^4 = C_5^5 = 1$, так что всё сходится.

Итог:

$$(fg)^{(5)} = \sum_{k=0}^{5} C_5^k  g^{(k)} f^{(5-k)}$$

— то, что нужно.

## Переход к общему $n$

Та же логика для произвольного $n$:

- $k = 0$: только из $S_2$, коэффициент $C_n^0 = C_{n+1}^0$.
- $k = n+1$: только из $S_1$, коэффициент $C_n^n = C_{n+1}^{n+1}$.
- $k = 1, \dots, n$: обе суммы, коэффициент $C_n^{k-1} + C_n^k = C_{n+1}^k$ (Паскаль).

Собираю:

$$(fg)^{(n+1)} = C_{n+1}^0  g^{(0)} f^{(n+1)} + \sum_{k=1}^{n} C_{n+1}^k  g^{(k)} f^{(n+1-k)} + C_{n+1}^{n+1}  g^{(n+1)} f^{(0)}$$

Края имеют тот же вид $C_{n+1}^k , g^{(k)} f^{(n+1-k)}$, поэтому заходят в сумму:

$$(fg)^{(n+1)} = \sum_{k=0}^{n+1} C_{n+1}^k  g^{(k)} f^{(n+1-k)}$$

Шаг индукции завершён. ∎

## Ошибки / тупики

- Застрял после разбиения на $S_1, S_2$ — не видел, что делать дальше. Подсказкой было то, что у сумм разная структура: $g^{(k+1)}$ против $g^{(k)}$. Это и есть сигнал, что нужен сдвиг.
- Сначала казалось, что сдвиг меняет слагаемые. На самом деле — нет, это переименование индекса. Проверка вручную при $n=2$ убедила: до сдвига и после — одни и те же слагаемые, просто пронумерованы по-другому.
- Не понимал, как складывать суммы с разными пределами. Помогло выписать столбиком при конкретном $n$: сразу видно, что края одиночные, серединка парная.

## Инсайт

Сдвиг индекса суммирования — это переименование, а не изменение. Сумма $\sum_{k=0}^{n} a_{k+1}$ и сумма $\sum_{k=1}^{n+1} a_k$ — одно и то же, просто слагаемые «пронумерованы» иначе.

Связка **индукция + сдвиг индекса + Паскаль** — стандартный шаблон для формул, выглядящих как биномиальные. На этой же схеме держится доказательство самого бинома Ньютона; формула Лейбница — его аналог для производных.

## Итог

- Доказательство — индукция по $n$.
- Главный приём — сдвиг индекса в одной сумме, чтобы выровнять структурную часть слагаемых.
- Финал — Паскаль $C_n^{k-1} + C_n^k = C_{n+1}^k$.
- Если запутался в суммах — выписать столбиком при малом $n$.

## Связанные задачи

- Бином Ньютона $(a+b)^n = \sum C_n^k a^k b^{n-k}$ — та же схема доказательства.
- $n$-я производная конкретных произведений: $(x^2 e^x)^{(n)}$, $(x \sin x)^{(n)}$ — прямое применение Лейбница.
- Тождество $\sum_{k=0}^{n} C_n^k = 2^n$ — бином при $a = b = 1$.
- Формула Фаа-ди-Бруно — обобщение для $n$-й производной композиции.
